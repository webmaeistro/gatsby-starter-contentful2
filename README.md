# Gatsby Contentful Starter

A starter for Gatsby using Contentful as CMS with optional deploy to Netlify.

![Home Screen](/src/images/homescreen.png?raw=true "Home Screen")

- Markup

[Gatsby](https://www.gatsbyjs.org)
